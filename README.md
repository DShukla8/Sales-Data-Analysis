# Sales Analysis Dashboard (Python + Power BI)
## Tools Used
- Python (Pandas)
- Power BI

## Project Description
This project analyzes sales data to identify trends and patterns. Data cleaning and preprocessing was performed using Python (Pandas), and an interactive dashboard was developed in Power BI to visualize key business insights.

## Key Insights
- Sales by Region and Category
- Monthly Sales Trend
- KPI analysis
- Identified top-performing regions and product categories
